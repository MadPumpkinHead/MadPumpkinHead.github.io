# Meine einfache Website

Dies ist eine minimal gehaltene, professionelle, aber leicht verspielte persönliche Website für Arshiya Mozafari.

Wichtige Hinweise:
- Die Seite enthält die öffentliche E‑Mail-Adresse `i.mozafari@sus.bruelmatt.ch`. Du hast "ok publish" bestätigt.

Wie du lokal bearbeitest:
1. Git installieren und das Repo klonen:
   ```
   git clone https://github.com/MadPumpkinHead/MadPumpkinHead.github.io.git
   cd MadPumpkinHead.github.io
   ```
2. Dateien bearbeiten: `index.html`, `styles.css` und `assets/profile.png`.
3. Commit und push:
   ```
   git add index.html styles.css assets/profile.png
   git commit -m "Website: add profile, styles, and contact"
   git push
   ```

Deployment mit GitHub Pages:
- Da das Repo `MadPumpkinHead.github.io` heißt, wird die Seite automatisch veröffentlicht vom `main` branch unter: `https://MadPumpkinHead.github.io`.
